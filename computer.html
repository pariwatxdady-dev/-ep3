คอมพิวเตอร์คืออะไร

คอมพิวเตอร์ (Computer) คืออุปกรณ์อิเล็กทรอนิกส์ที่สามารถรับข้อมูล ประมวลผลข้อมูล และแสดงผลลัพธ์ออกมาได้ โดยใช้โปรแกรมคำสั่งในการควบคุมการทำงาน

ส่วนประกอบของคอมพิวเตอร์

  • CPU – หน่วยประมวลผลกลาง
  • RAM – หน่วยความจำชั่วคราว
  • Storage – พื้นที่เก็บข้อมูล เช่น SSD หรือ HDD
  • Input Device – อุปกรณ์รับข้อมูล เช่น เมาส์ คีย์บอร์ด
  • Output Device – อุปกรณ์แสดงผล เช่น จอภาพ

ประเภทของคอมพิวเตอร์

1. Desktop Computer 2. Laptop 3. Tablet 4. Smartphone 5. Server

ประโยชน์ของคอมพิวเตอร์

- ใช้ทำงานเอกสาร - ใช้เรียนออนไลน์ - ใช้เขียนโปรแกรม - ใช้เล่นเกม - ใช้ทำกราฟิกและวิดีโอ
